Тестовое задание для Junior PHP разработчика.

Описание приложения
Сервис каталога книг.
Каталог будет содержать объекты:
Книга
Автор
Книга
Название
Описание
Год написания
Жанр (список тегов)
Автор (может быть написана в соавторстве)
Автор
ФИО автора
Функции каталога
Книга
Вывод списка книг
Вывод списка книг одного автора
Вывод списка книг нескольких жанров
Вывод списка книг написанных за определенный период
Вывод информации по книге
Добавление книги
Изменение книги
Автор
Вывод списка авторов
Добавление автора
Изменение автора
Удаление автора

Взаимодействие с сервисом посредством HTTP запросов к API. Все ответы представляют собой JSON объекты



Задание:
Реализовать API
Подготовить тестовые данные
Написать краткую документацию по использованию сервиса
Список методов API, с кратким описанием
Список параметров для каждого метода
Описание параметров ответа для методов
Опубликовать приложение, либо подготовить Docker контейнер
Предоставить ссылку на публичный репозиторий.

Требования:
Решение выполнить используя фреймворк Symfony, Yii2, либо без использования фреймворков
Веб сервер Nginx
База данных MySql
Код оформляется по стандартам PSR
